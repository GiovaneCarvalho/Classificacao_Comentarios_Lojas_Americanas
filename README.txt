O Modelo a seguir é construido em cima de comentários feitos por usuários no site da Americanas

A base de dados está disponível no formato parquet (Mais leve). Utilize ela como base do algorimto, deixando o arquivo na mesma pasta. Você pode baixa-la no link: https://drive.google.com/drive/folders/1FTXF1AGRvwxtjozPx0zvY4XbygyWdG23?usp=sharing

O Objetivo do modelo é realizar a classificação binária, predizendo a partir do feedback o sentimento a respeito do comentário, sendo positivo ou negativo. A base de dados disponível é uma base curada e resultante do estudo e tratamento realizados no modelo da abordagem A.

É recomendado que a execução seja feita em um servidor com poder computacional consideravel, como sessões do Colab com GPU disponível.

O Modelo a seguir é construido em cima de comentários feitos por usuários no site da Americanas

A base de dados está disponível no formato parquet (Mais leve). Utilize ela como base do algorimto, deixando o arquivo na mesma pasta. Você pode baixa-la no link: https://drive.google.com/drive/folders/1LdsIb7jii7nzPvtOYW5miYgLrDAgDlqD?usp=sharing

O Objetivo do modelo é realizar a classificação multiclasses, predizendo a partir do feedback a nota dada pelo usuário. Esta nota varia de 1 a 5. Além disso, o conteúdo do Jupyter apresenta uma análise exploratória dos dados, dando informações importantes sobre o problema.

É recomendado que a execução seja feita em um servidor com poder computacional consideravel, como sessões do Colab com GPU disponível.
